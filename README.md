# VRP (AI assignment)

University project on Artificial Intelligence that solves the **Vehicle Routing Problem (VRP)** using **KNN** and a **genetic algorithm**. The system works with graphs, prioritized and sized packages, planning efficient routes for a fleet of vehicles.

## 📂 Project Structure

```
.
├── data/          # CSV files with packaging, vehicles, distances and office location
├── src/           # Coding
├── main.py        # Main entry point of the project
```

## ⚙️ Requirements

- Python   
- Libraries (you must intall it with pip command): pandas and scikit-learn
